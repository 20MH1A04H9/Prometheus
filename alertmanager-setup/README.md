# 📖 Overview

:contentReference[oaicite:2]{index=2} handles alerts generated by :contentReference[oaicite:3]{index=3}.

When Prometheus detects conditions like:

- High CPU Usage  
- High Memory Usage  
- Disk Full

- Email  
- Slack

# ⚙ Install Alertmanager

Download Alertmanager:

[Alertmanager Releases](https://github.com/prometheus/alertmanager/releases?utm_source=chatgpt.com)

```bash
cd /tmp
wget https://github.com/prometheus/alertmanager/releases/latest/download/alertmanager-0.27.0.linux-amd64.tar.gz
```

Extract:

```bash
tar xvf alertmanager-0.27.0.linux-amd64.tar.gz
cd alertmanager-0.27.0.linux-amd64
```

Move binaries:

```bash
sudo cp alertmanager /usr/local/bin/
sudo cp amtool /usr/local/bin/
```

Create directories:

```bash
sudo mkdir /etc/alertmanager
sudo mkdir /var/lib/alertmanager
```

Copy configuration:

```bash
sudo cp alertmanager.yml /etc/alertmanager/
```

---

# 🔧 Create Alertmanager Service

```bash
sudo nano /etc/systemd/system/alertmanager.service
```

Paste:

```ini
[Unit]
Description=Alertmanager
After=network.target

[Service]
User=prometheus
Group=prometheus

ExecStart=/usr/local/bin/alertmanager \
 --config.file=/etc/alertmanager/alertmanager.yml \
 --storage.path=/var/lib/alertmanager

Restart=always

[Install]
WantedBy=multi-user.target
```

# ▶ Start Alertmanager

```bash
sudo systemctl daemon-reload
sudo systemctl enable alertmanager
sudo systemctl start alertmanager
sudo systemctl status alertmanager
```

# ⚙ Configure Prometheus

Edit Prometheus config:

```bash
sudo nano /etc/prometheus/prometheus.yml
```

Add:

```yaml
alerting:
  alertmanagers:
    - static_configs:
        - targets:
          - "localhost:9093"
```

Restart Prometheus:

```bash
sudo systemctl restart prometheus
```

# 🚨 Create Alert Rules

Create rule files:

```bash
sudo nano /etc/prometheus/alert.rules.yml
```

For Windows alerts:

```bash
sudo nano /etc/prometheus/windows-alerts.yml
```

---

## Linux Memory Alert Example

```yaml
groups:
  - name: system-alerts
    rules:
      - alert: HighMemoryUsage
        expr: clamp_min(
          (1 - (node_memory_MemAvailable_bytes / node_memory_MemTotal_bytes)) * 100,
          0
        ) > 80
        for: 2m
```

---

## Windows Alert Examples

### High Memory

```yaml
alert: WindowsHighMemoryUsage
```

### High CPU

```yaml
alert: WindowsHighCPUUsage
```

### High Disk Usage

```yaml
alert: WindowsHighDiskUsage
```

### Service Down

```yaml
alert: WindowsServiceDown
```

### Unexpected Reboot

```yaml
alert: WindowsUnexpectedReboot
```
